# Build
```bash
npm install
npm run build
```

After build, the `dist` directory will be generated, and the directory structure is as follows:

```
dist
├── NebulaGraph-Cloud
│   └── svg.tpl
├── NebulaGraph-Dashboard
│   └── svg.tpl
├── NebulaGraph-Explorer
│   └── svg.tpl
├── NebulaGraph-Interface-Guidelines
│   └── svg.tpl
├── NebulaGraph-Studio
│   └── svg.tpl
└── NebulaGraph-Website
    └── svg.tpl
```

# Publish
```bash
# npm publish --tag beta
npm publish
```
